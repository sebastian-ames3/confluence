## Objective

Interpret charts, volatility surfaces, and technical analysis images from Discord and Twitter.

---

## Success Criteria

- [ ] OCR extracts text from images
- [ ] Claude vision interprets charts
- [ ] Extracts data from volatility surfaces
- [ ] Identifies technical levels
- [ ] Sebastian validates 10 charts with >85% accuracy

---

## Agent Specification

### Input
```python
{
    "image_path": "/path/to/chart.png",
    "source": "discord|twitter",
    "context": "SPX volatility surface update"
}
```

### Output
```python
{
    "image_type": "volatility_surface|technical_chart|positioning_chart",
    "extracted_text": "OCR text...",
    "interpretation": {
        "main_insight": "VIX term structure inverted",
        "key_levels": ["SPX 5800 support", "6000 resistance"],
        "implied_volatility": {"30d": 0.18, "60d": 0.16},
        "positioning": "heavy call buying Dec expiry"
    },
    "tickers": ["SPX", "VIX"],
    "sentiment": "bearish"
}
```

Uses Claude vision API for chart interpretation, pytesseract for OCR.
